Introduction

Ada Lovelace is the first one to invent a programming language which was an algorithm in 1883. Charles Babbage’s Analytical engine was the reason that made Ada Lovelace created this algorithm. This algorithm was made in order to compute Bernoulli numbers.

The computer codes were specialized for the first time by Alonzo Church and Alan Turing in 1936 when Church succeeded in expressing calculus of lambda in a formulaic way.


The first modern computers powered by electricity were invented in the 1940s, but these computers' speed and memory were low. Due to lack of resources, programmers had to manually write completed language programs. Programing low-level programming language manually was difficult. Meanwhile, programmers realized that high-level languages are required to reduce the amount of effort made needed to write completed language programs manually.


Konrad Zuse tried to invent the first-ever high-level languages. He developed the programming language Plankalkül between 1943 and 1945, but he didn't complete it for some reason.

The first time a high-level programming language was invented for electronic computers was in 1949. John Mauchly suggested Short Code which was originally known as brief code, and then William F Schmidt it completed it. It was the first programming language used practically

Short Code was represented in a mathematical expression that can be understood by humans easily, but the program has to be translated into the machine code every time it ran which was considered as a disadvantage as it consumes many time. In 1952, Alick Glennie invented Autocode. It was the first language that was compiled where it is directly converted into machine code by using a compiler. 

Two years later, R.A. Brooker developed the second version which was Mark 1 auto code.
Autocode was better than the previous programming languages, but it was not popular. A team in IBM developed FORTRAN which was the first well-known programming language in 1954. It is considered as the oldest high-level programming language which is still used until now.

High-level languages such as Java, C, and C++ have had still time to come out. But they had been developed for the first time in 1958 by a group of scientists for the purpose of research and scientific uses. They succeeded in developing a language known as Algol where Java and other high-level languages considered it as a base.
